⏱️ Duración ideal
10-15 minutos por sesión. Parar ANTES de que se canse, siempre con final positivo.
🎯 Una instrucción a la vez
Esperar que ejecute cada instrucción antes de dar la siguiente. Nunca varias juntas.
🔊 Pronunciar el SONIDO, no el nombre
No decir "eme" — decir /m/ como el sonido real. Los nombres de letras confunden.
🏆 Corrección positiva siempre
"¡Casi! Era /s/ de sapo 🐸" es mejor que "no, incorrecto". Mantener el tono de juego.
🚗 Las Misiones del Auto son las más valiosas
Sin pantalla, el canal auditivo trabaja más. El movimiento tiene efecto regulador en el sistema nervioso. Ideal para el notch 4kHz de Mateo.
🎧 Considerar el oído de Mateo
Mateo tiene mayor dificultad con fricativas (/s/, /f/, /z/) especialmente en el oído derecho. Pronunciarlas más despacio, hablar cerca de su oído izquierdo.
📅 Frecuencia
4-5 veces por semana, 10-15 min. La constancia supera la intensidad.
📊 Leer el historial
Una precisión creciente semana a semana es la señal más importante. No comparar sesiones aisladas. Compartir pantalla "Historial" con la fonoaudióloga.

📖 Cómo configurar Supabase (gratis)

1. Crear cuenta en supabase.com

2. Crear un nuevo proyecto

3. En el Editor SQL, ejecutar:

create table sesiones (
  id         bigserial primary key,
  created_at timestamptz default now(),
  fecha      bigint,
  correct    integer default 0,
  wrong      integer default 0,
  total      integer default 0,
  pct        integer default 0,
  stars      integer default 0,
  streak     integer default 0,
  games      integer default 0,
  duration   bigint  default 0
);
alter table sesiones enable row level security;
create policy "public read"   on sesiones for select using (true);
create policy "public insert" on sesiones for insert with check (true);
create policy "public delete" on sesiones for delete using (true);

4. Ir a Settings → API

5. Copiar Project URL y anon public key
